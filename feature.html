Linux Mint の特徴

Linux Mint のなにがすごいの?Linux Mint は何ができるの?とか思っている人もいるでしょう。そういう人達のためにすぐに分かる Linux Mint の特徴をまとめてみました。

すべてのソフトウェアが無料です

30000 を超えるソフトがすべて利用可能です

Linux Mint は非常にセキュリティ性が強く、ウィルスに感染することは心配しなくていいでしょう

生産性、安定性と成熟した MATE デスクトップ

MATE デスクトップはシンプルで使いやすく、安定性と生産性に長けています。そして低性能なパソコンでもサクサク動きます。

現代的で新しくエキサイティングな Cinnamon デスクトップ

Cinnamon デスクトップは高性能で先進的な技術を持ち合わせています。ユーザーに新たな刺激と体験を与えます。

高機能な Office アプリケーション

Linux Mint は高機能な Office アプリケーションが付属しています。Microsoft Office 2003 とよく似たインターフェースを持っていて、Microsoft Office 2010 や PDF を読み込むことができ、編集もできます。もちろん Microsoft Office 2010 や PDF 形式に出力することも可能です。

利用可能なアプリケーションとしては Writer という文書作成ソフトウェアや、Calc という表計算ソフト、Impress というプレゼンテーションのソフトも利用可能です。ほかにも Draw や Base、Math などがあります。これらは Microsoft Office と互換性を意識しているので新たに技術を習得する必要はほとんど無いでしょう。

マルチメディアにも対応していて、DVD や音楽が視聴可能です

高機能なメディアプレーヤーの VLC や、Banshee などのソフトウェアが付属しているため簡単に DVD や音楽などが視聴できます。

画像の加工、イラストの作成もできます

Inkscape や GIMP などといったアプリケーションがあるため DTP やイラスト作成ができます。機能も多いため複雑な画像処理やイラストの作成などができます。

これらのアプリケーションは書籍も多いので簡単に初めての方でも学習できます。

数々の DAW ソフトウェアやソフトシンセにより簡単に、無料で DTM ができます。

高機能な DAW ソフトウェア、Ardour や Rosegarden などがあり、ソフトシンセも YAMAHA DX7 のエミュレータの Hexer などがあるため DTM が無料で出来ます。

簡単なソフトウェア管理

Linux Mint にはすばらしいソフトウェア管理ツールが含まれているため、Windows のように新しいソフトをいちいち Web から探してダウンロードして、インストールする必要はありません。Linux Mint の場合は管理ツールから欲しいソフト名を入力したり、ジャンルから目的のソフトウェアを探し出したりしてインストールができます。

ユーザーによるヘルプとサポートも受けられます

Linux Mint Japan にはフォーラムや Wiki やメーリングリスト等いろいろな交流する場があります。そのため質問や困ったことがあれば気軽に投稿できます。
