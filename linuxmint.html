Linux Mint とは

Linux Mint の目的

Linux Mint の目的は最新で、洗練されて、簡単に使える、強力で使いやすいオペレーティングシステムを製作することです。2006 年から開発が始まった Linux Mint は今や Microsoft 社の Windows、Apple 社の Mac OS、Canonical 社の Ubuntu に続いて 4 番目に広く使われている家庭用オペレーティングシステムです。

Linux Mint 成功の秘訣の一部:

Linux Mint は難しい設定が不要で、マルチメディアを完全にサポートし極めて簡単に使えます。(ただし Linux Mint 日本語化版は nocodecs 版をベースとしているためマルチメディアを完全にサポートはしていません)

Linux Mint は無償で、オープンソースです。

Linux Mint はコミュニティが開発を主導するディストリビューションです。ユーザがプロジェクトにフィードバックを送ることを奨励しているため、ユーザのアイディアが Linux Mint を 改善するために使われることがあります。Linux Mint は Debian と Ubuntu をベースとしておりおよそ 30,000 個のパッケージと最高のソフトウェアマネージャを提供します。Linux Mint は安全で信頼できます。慎重なソフトウェアアップデート、独自のアップデートマネージャ、堅牢な Linux アーキテクチャを採用することで Linux Mint はほぼメンテナンスが要りません (アップデートに伴う不具合、ウィルス、スパイウェア等への対策が不要です)。
