Linux Mint Japan へのお問い合わせ

Linux Mint、あるいは Linxu Mint Japan にお問い合わせをしたい場合は webadmin@linuxmint-jp.net にメールを下さい。

IRC チャンネル

Linux Mint Japan の IRC チャンネルは非常にフレンドリーで気軽に会話できます。Linux Mint に関する話題でも雑談でもいいので手軽に話しあいましょう。 クライアントは Linux では XChat や Pidgin や Quassel IRC があり、Windows では Pidgin や Lime Chat などがあります。

チャンネル: freenode 上の #linuxmint-help-jp

freenode の irc に参加するには web インターフェイスを用いる方法もあります。http://webchat.freenode.net/?channels=linuxmint-help-jp
